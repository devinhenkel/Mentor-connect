# Mentor Connector (prototype)

Next.js + Tailwind prototype for LCP mentor discovery, theme browsing, and a simulated “check-out” flow.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS v4 (`src/app/globals.css` theme tokens)
- Primary brand color: `#FF6720` (defined as `primary` in Tailwind theme)
